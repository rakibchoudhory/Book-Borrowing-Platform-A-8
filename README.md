# 📚 Online Book Borrowing Platform

## Purpose

Online Book Borrowing Platform is a modern web application designed to simplify the process of discovering, borrowing, and managing books online. The platform enables users to explore available books, borrow them with ease, and manage their profiles through a secure and user-friendly interface.

## Live URL

🔗 https://online-book-borrowing-platfrom.vercel.app

## Key Features

* Secure user authentication and authorization using Better Auth.
* Browse and explore the collection of available books.
* Borrow and return books through an intuitive workflow.
* Manage user profiles and update personal information.
* Protected routes to ensure authorized access only.
* Responsive and user-friendly interface for all devices.
* Interactive UI components and smooth user experience.
* Toast notifications for real-time feedback on user actions.
* Modern animations and sliders to enhance user engagement.

## Installation and Setup

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project directory:

```bash
cd online-book-borrowing-platfrom
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Start the production server:

```bash
npm start
```
